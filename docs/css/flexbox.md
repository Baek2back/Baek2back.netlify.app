# flexbox

## flexbox의 두 개의 축

주축은 `flex-direction` 속성을 사용하여 지정하며 교차축은 이에 수직인 축으로 결정된다.

### 주축

주축은 `flex-direction`에 의해 정의된다.

- `row`
- `row-reverse`
- `column`
- `column-reverse`
